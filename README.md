# Telegram Channel Bot

Бот для отправки сообщений в Telegram-канал. Только владелец (ты) может управлять ботом.

## Деплой на Railway

### 1. Подготовка

- Создай бота через [@BotFather](https://t.me/BotFather) → `/newbot` → скопируй токен
- Узнай свой Telegram ID через [@userinfobot](https://t.me/userinfobot)
- Добавь бота администратором в свой канал (права: отправка сообщений, закрепление)

### 2. Деплой

1. Залей проект на GitHub
2. Зайди на [railway.app](https://railway.app) → **New Project** → **Deploy from GitHub repo**
3. Выбери репозиторий

### 3. Переменные окружения

В Railway перейди в **Variables** и добавь:

| Переменная    | Пример значения          | Описание                        |
|---------------|--------------------------|---------------------------------|
| `BOT_TOKEN`   | `1234567890:ABCdef...`   | Токен от @BotFather             |
| `YOUR_USER_ID`| `987654321`              | Твой числовой Telegram ID       |
| `CHANNEL_ID`  | `@mychannel`             | Username или `-1001234567890`   |

### 4. Готово

Railway автоматически запустит бота. Статус смотри во вкладке **Deployments**.

## Команды бота

| Команда | Описание |
|---|---|
| `/send Текст` | Отправить текст в канал |
| `/sendbuttons Текст` | Отправить текст с inline-кнопками |
| `/forward` | Переслать следующее сообщение/фото/видео |
| `/photo` | Отправить фото с подписью |
| `/pin 123` | Закрепить сообщение по ID |

Поддерживается HTML-разметка: `<b>жирный</b>`, `<i>курсив</i>`, `<code>код</code>`

## Как добавить свои кнопки

Открой `bot.py`, найди раздел `sendbuttons` и измени:

```python
markup.row(
    InlineKeyboardButton("📢 Прямой эфир", url="https://t.me/твоя_ссылка"),
    InlineKeyboardButton("🔵 Наш MAX",     url="https://t.me/твоя_ссылка2")
)
```
